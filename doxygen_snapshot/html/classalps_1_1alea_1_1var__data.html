Detailed Description

template<typename T, typename Strategy = circular_var>
class alps::alea::var_data< T, Strategy >

Data for variance accumulation.

As with mean_acc, this class is basically a "union"-like structure, which for a data series (X[0], ... X[count_-1]) either represents the sum of X[i] and the sum of X[i]*X[i] (sum state) or the sample mean and sample variance of X (mean state).
